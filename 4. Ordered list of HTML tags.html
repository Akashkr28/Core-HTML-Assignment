List of HTML Tags

  1. html Tag: The root element of an HTML document.
  2. head Tag: Contains metadata about the document, such as the title and linked stylesheets.
  3. title Tag: Sets the title of the web page, displayed in the browser's title bar or tab.
  4. body Tag: Contains the visible content of the web page, including text, images, and other elements.
  5. h1 Tag to h6 Tag: Headings used to define the hierarchy of content.
  6. p Tag: Represents a paragraph of text.
  7. a Tag: Defines a hyperlink, allowing users to navigate to another web page or resource.
  8. img Tag: Embeds an image on the web page.
  9. ul Tag: Creates an unordered (bulleted) list.
  10. ol Tag: Creates an ordered (numbered) list.
  11. li Tag: Represents a list item within <ul> or <ol>.
  12. div Tag: A generic container for grouping and styling content.
  13. span Tag: A generic inline container for styling a portion of text.
  14. form Tag: Used to create HTML forms for user input.
  15. input Tag: Defines an input field within a form.
